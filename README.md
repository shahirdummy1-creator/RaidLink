# PAYANUM — Taxi Booking App

A full-stack taxi booking platform built with Flask and MySQL.

## Features
- Rider signup, login and booking
- Driver signup, login and trip management
- Real-time booking requests for drivers
- Admin dashboard with driver and rider management
- OTP-based trip start verification
- Document upload for drivers

## Stack
- **Backend:** Python / Flask
- **Database:** MySQL
- **Frontend:** Bootstrap 5
